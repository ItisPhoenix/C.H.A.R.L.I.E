# Report Writing Guide

## Purpose
Standards and templates for clear, professional report writing.

## General Principles
- Lead with the conclusion/executive summary
- Use clear, concise language
- Structure with headers and bullet points
- Include evidence and citations
- End with actionable recommendations

## Report Structure

### Executive Summary
- One paragraph: what was done, key findings, recommendations
- Should stand alone without reading the full report

### Introduction/Background
- Context and scope
- Objectives
- Methodology used

### Findings/Results
- Organized by topic or severity
- Each finding: description, evidence, impact
- Use tables and figures where helpful

### Recommendations
- Specific, actionable steps
- Prioritized by impact/effort
- Include timelines where appropriate

### Appendices
- Raw data, detailed technical findings
- Glossary of terms
- References and citations

## Writing Style
- Active voice preferred
- Short paragraphs (3-5 sentences)
- Avoid jargon without explanation
- Use consistent terminology throughout
