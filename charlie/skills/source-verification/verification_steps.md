# Source Verification Steps

## Purpose
Systematic fact-checking workflow to ensure information accuracy.

## Verification Process

### Step 1: Claim Identification
- Extract specific factual claims from the source
- Separate facts from opinions
- Note the source's confidence level

### Step 2: Independent Verification
- Search for the same claim from independent sources
- Look for official/primary sources
- Check if the claim has been debunked

### Step 3: Context Check
- Verify the claim is not taken out of context
- Check if the source has a history of accuracy
- Look for corrections or retractions

### Step 4: Cross-Reference
- Compare with known facts
- Check against official documentation
- Verify statistics and numbers

### Step 5: Confidence Scoring
- **Confirmed**: Multiple independent sources agree
- **Likely**: 2+ sources agree, no contradictions
- **Uncertain**: Single source, cannot verify
- **Disputed**: Contradicting sources found
- **Debunked**: Evidence contradicts the claim
